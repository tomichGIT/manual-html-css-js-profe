Flexbox en 15min

Opciones de container:
  1. Flexbox me permite editar los estilos de los elementos desde dentro del container, sin tener que convertir en Flex a cada item
  2. Solo por utilizar display:flex ya todos mis items van a tener el mismo alto y se van a alinear de izq a der (row)
  3. Flex direction me va a permitir decidir si trabajo en una Row o una Column
Opciones de items:
  1. Los items también se pueden modificar. Flex-shrink=0 indica que no debe achicarse. Flex-grow=1 indica que debe ocupar mas espacio
  2. Para manejar el monto de grow y shrink puedo elegir desde que ancho comenzar usando flex-basis
  3. Puedo usar un shorthand de grow shrink y basis con "flex: 0 1 25px";
  4. Cada item puede tener el valor "order" para re-ordenar los items (no recomendados para Tab y eReaders)

Ejemplo 1

En este ejemplo, se muestran los atributos de Flexbox:

  • display: flex - Activa el modelo de caja flexible en el contenedor.
  • justify-content: space-between - Distribuye los elementos a lo largo del eje principal con espaciado uniforme entre ellos.
  • align-items: center - Centra los elementos a lo largo del eje cruzado.
  • flex-direction: row - Establece la dirección principal como horizontal (fila).
  • flex-wrap: wrap - Permite que los elementos se envuelvan a la siguiente fila cuando no caben en el contenedor.

Este ejemplo ilustra cómo la propiedad flex-grow afecta el crecimiento de elementos en un contenedor.
